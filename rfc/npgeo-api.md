# NPGEO Corona 

## Media-Type
    
```
application/npgeo.hospital.location.api+json
```
    
## Syntax

```
contact_website (string)
name (string)
amenity (string)
address_full (string)
rooms (number)
beds (number)
capacity (number)
wheelchair (string)
emergency (string)
geometry (array(geometry)
zeitpunkt (date)

geometry =
    x (number)
    y (number)
```
